# `vikiclow setup`

Initialize `~/.vikiclow/vikiclow.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
vikiclow setup
vikiclow setup --workspace ~/.vikiclow/workspace
```

To run the wizard via setup:

```bash
vikiclow setup --wizard
```
